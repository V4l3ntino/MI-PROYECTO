Para saber como funciona html lo primero que debemos saber es que este es un lenguaje de marcas, es decir que todo lo que queramos ver en la página web se tiene que especificar con etiquetas. Html nos sirve para crear el esqueleto de nuestra página web, es decir lo más básico que es el texto que queramos escribir, los enlaces que le vayamos a meter, videos ... Pero luego si queremos que nuestros datos se vean sobre un fondo bonito y posicionar el texto a nuestro gusto pues para eso utilizaremos las hojas de estilo css.

Recomiendo que si vas a empezar en esto utilices Visual Studio code porque es mucho más comodo a la hora de escribir etiquetas. Casi todas las etiquetas tienen que siempre abrirse y cerrarse al final de la siguiente manera:

por ejemplo la etiqueta p sirve para escribir un párrafo de texto, entre estas etiquetas será donde nosotros podemos escribit todo el texto que queramos.

La etiqueta "div" sirve como contenedor, es decir que es como una caja donde nosotros podemos meterle otros elementos como puede ser párrafos o imágenes. Dicho a groso modo esta etiqueta nos sirve para encapsular otros elementos.

La etiqueta header y footer: estas nos sirven para crear cabeceras y pies de páginas.

La etiqueta ol: podemos crear listas numeradas, para ello dentro de estas etiquetas debemos meter otra etiqueta denominada li. Una manera de recordarlo esque "ol" es la abreviación de "Orden List"

La etiqueta ul: podemos crear listas no numeradas, esta sigue la mima estructura que las listas numeradas.

Si queremos meter una imagen podemos utilizar la etiqueta img, dentro de esta debemos meterle dirección de la ruta donde se encuntra nuestras imágenes. Siempre debemos utilizar rutas relativas porque sino nos dará problemas a la hora de subir nuestra página web a un hosting. Esto se debe a que el la estructura de los directorios de nuestra computadora no es la misma a la estructura de un servidor.
