Como empezar en HTML

HTML Y CSS
HTML: para saber como funciona html lo primero que debemos saber es que este es un lenguaje de marcas, es decir que todo lo que queramos ver en la página web se tiene que especificar con etiquetas. Html nos sirve para crear el esqueleto de nuestra página web, es decir lo más básico que es el texto que queramos escribir, los enlaces que le vayamos a meter, videos, etc.
CSS: las hojas de estilo son líneas de código HTML o XML que están orientadas a darle un estilo específico a los elementos de un documento HTML, es decir, se utilizan para personalizar el color, los backgrounds, las fuentes, los enlaces y todo el apartado visual de un sitio web, con la finalidad de mantener lo más limpio posible el código fuente de la página web.

Una estructura HTML se empieza con la etiqueta "html" y acaba con " /html". Ambas etiquetas se deben poner dentro del signo mayor y menor <>.
Todo lo que esté en medio será la página web. Dentro de "html" se encuentran 2 partes diferenciadas. La primera "head" es la cabecera de la página. Aquí irán cierta información que no es directamente el contenido de la página. Aquí se pone el título de la página, los metadatos, estilos, código javascript (todo esto se estudiará en capítulos venideros). La primera que se suele estudiar es "title" , que indica el título de la página (lo que el navegador pone en la parte superior izquierda).

La segunda parte es "body""/body"(lo pongo entre comillas para que se muestre en pantalla, vosotros teneis que ponerlo entre <>). Aquí va propiamente el contenido de la página: fotos, párrafos, formularios, etc. Por ejemplo, siguiendo con el ejemplo de la página anterior, el siguiente código, podemos cambiar el título de la página.

Tipos de elementos

La etiqueta p sirve para escribir un párrafo de texto, entre estas etiquetas será donde nosotros podemos escribir todo el texto que queramos. En visual studio podemos rellenar con texto escribiendo Lorem y dándole al tabulador.

La etiqueta "div" sirve como contenedor, dentro podemos meterle otros elementos como puede ser párrafos o imágenes. Dicho a groso modo esta etiqueta nos sirve para encapsular otros elementos.

La etiqueta header y footer: estas nos sirven para crear cabeceras y pies de páginas.

La etiqueta ol: podemos crear listas numeradas, para ello dentro de estas etiquetas debemos meter otra etiqueta denominada li. Una manera de recordarlo esque "ol" es la abreviación de "Orden List" y "li" de lista.

Si queremos meter una imagen podemos utilizar la etiqueta img, dentro de esta debemos meterle la dirección de la ruta donde se encuntra nuestras imágenes. Siempre debemos utilizar rutas relativas porque sino nos dará problemas a la hora de subir nuestra página web a un hosting. Esto se debe a que la estructura de los directorios de nuestra computadora no es la misma a la estructura de un servidor.

Con la etiqueta table podemos crear tablas, dentro de esta etiqueta se utiliza otros dos elementos que son "td" y "tr". Con "td" creamos las columnas y con "tr" las filas. También se utiliza "th" que sirve para crear una fila como cabezera. Aquí os muestro un ejemplo.
