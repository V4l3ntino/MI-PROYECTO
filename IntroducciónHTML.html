Para saber como funciona html lo primero que debemos saber es que este es un lenguaje de marcas, es decir que todo lo que queramos ver en la página web se tiene que especificar con etiquetas. Html nos sirve para crear el esqueleto de nuestra página web, es decir lo más básico que es el texto que queramos escribir, los enlaces que le vayamos a meter, videos ... Pero luego si queremos que nuestros datos se vean sobre un fondo bonito y posicionar el texto a nuestro gusto pues para eso utilizaremos las hojas de estilo css.
